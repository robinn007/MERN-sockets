
## How to Run Locally

1. Clone the repository:

   ```bash
   git clone https://github.com/robinn007/MERN-sockets.git

   ```
2. cd server
3. nvm install 12.22.12
4. nvm use 12.22.12
5.  npm install
6.  To start the server
   ```bash
   npm run dev
   ```
